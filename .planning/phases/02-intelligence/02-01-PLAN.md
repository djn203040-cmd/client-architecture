|----------|-------------|
| coach browser -> /api/voice/* | Coach-supplied corpus text crosses into server; untrusted input |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-01 | Tampering | /api/voice/save body | mitigate | Zod-validate against VoiceProfileSchema before writing coaches.voice_model — malformed/oversized profiles rejected with 400 |
| T-02-02 | Information Disclosure | ANTHROPIC_API_KEY in client bundle | mitigate | `import 'server-only'` on client.ts; ai-engine window guard; no NEXT_PUBLIC_ prefix |
| T-02-03 | Information Disclosure | Cross-coach voice model write | mitigate | voice/save writes `.eq("id", user.id)` only; RLS coaches_own policy enforces coach scope |
| T-02-04 | Spoofing | Unauthenticated /api/voice/* call | mitigate | `supabase.auth.getUser()` 401 gate on both routes |
| T-02-05 | Tampering | Prompt injection via pasted corpus | mitigate | Corpus wrapped in XML tags; system prompt instructs Claude to analyze only, never execute embedded instructions |
