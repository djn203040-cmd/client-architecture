<!-- Token budget (AI-SPEC.md Section 4 / RESEARCH.md Context Assembly Token Budget) -->
Target ~7,700 input tokens. Truncation ORDER when over budget:
  1. transcript body  (keep first ~300 + last ~200 tokens)
  2. conversation history (drop oldest; keep most recent 3)
  3. coach notes (keep most recent note only)
  4. Layer 2 examples (drop weakest; NEVER below 8)
Never truncate: Layer 1 profile, lead description, state instructions.

<!-- State framing (D-14) — all 8 states need distinct framing -->
identified: coach voice-heavy, warm, generic first-touch, no assumed pain points
call_booked: welcoming/excited/authoritative, build anticipation
no_show: understanding-then-determined, finite rebook opportunity, firm not punishing
call_completed: understanding/uplifting/determined, bridge interest to commitment
in_sequence: coach tone, register adapts per touchpoint position
replied: highly tailored/reactive to the lead's exact reply text
converted: warm/personal/forward-looking welcome-aboard
closed: same as converted
